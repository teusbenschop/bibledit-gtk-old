3 Web

Bibledit-Web is a web application for translating the Bible. It is part of the Bibledit suite.
A demo is available at http://bibleconsultants.dyndns.org/bibledit-web-demo. This is an open installation. You are always logged in as "admin". (A standard installation is secured through a login.)
More information is available at http://bibleconsultants.dyndns.org/bibledit-web-demo/help/index.php.
Translations are being done on https://launchpad.net/bibledit, your help is welcomed.
