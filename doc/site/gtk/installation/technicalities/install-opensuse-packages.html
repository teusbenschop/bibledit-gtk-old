Install openSUSE packages

1. Click the Suse icon, at the bottom left, then Computer, then Install/Remove Software.

2. Search for the first package given in the list of packages to install, and tick it.

3. Click button Accept.

4. In window "Changed Packages", click "Continue".

5. The installer will take a while to update the system. Close the window when through.

6. Go to step 2 to install the next package from the list, repeating this step till all packages have been installed.

7. After completion, close the window.
