Version 3.9

6 October 2009

A couple of bugs were fixed, one of them related to importing raw text.

The KJV Bible from the Sword project was made part of Bibledit.

The keyterms per verse and their renderings show again. The system was extended.

The window that shows the keyterms in a verse has clickable keyterms and Strong's numbers.

The references window has been redone. It now looks and feels like a web page.

The translator can display a window with related verses.

The related verses window shows the keyterms in the verse.

The related verses window shows the Strong's numbers in the verse and where these occur elsewhere in the Bible.

The related verses window shows the parallel passages in the Old and New Testament.

The references window shows the verse text of the active editor, and of the King James Bible.

The management of hidden references is now done from the references window itself.

The Quick References area was removed and its functionality was integrated into the References Window.

The buttons for go-back and go-forward got a history menu.

An Import Assistant was added replacing all the separate import routines.

The import of the kosher Yemenite Hebrew text produced by Mechon Mamre was moved to the Maintenance dialog.

Requires program "chmsee" for viewing .chm help files.

Works with BibleWorks 8, with older versions, and probably with newer ones too.

The Bibledit Windows Outpost code has been released under the GPL, and the compiled Windows executable is included with the package.

A couple of bugs in relation with the GoBibleCreator were fixed.

Honours the book selection during search or during replace.

The Bibledit Windows Outpost no longer shows on startup.

The search limits were fixed for search and for replace.

Can send references to the Online Bible.

Can receive the current reference from BibleWorks, and from Paratext and applications that can exchange references with it, and from the Online Bible.

Requires libdbus-glib-1-dev.

Can send verse references to Xiphos.

Can send verse references to and receive them from BibleTime.

Dialogs that are too tall for a small screen are automatically scaled to fit.

The "Verses" window has been removed. Its functionality is available much better in Xiphos, BibleTime, the Online Bible, and BibleWorks.

Dan created an autobuilder for windows builds, see http://buildbot.biblesint.org/waterfall.
