Version 3.8

11 July 2009

New undo engine.

The text editor got a redo function.

For faster navigation, it can remember the most recent verse per chapter, and most recent chapter per book.

A sample Resource was added.

Depends on libwebkit-1.0 for better display of web pages.

Drops libcurl dependency.

Easier Resource editor.

Snapshots were introduced speeding up Revert, Changes, Merge, and Backup.

On shutdown, the number of snapshots is trimmed based on their age.

The local git repository is used solely for collaboration.

A menu entry was added for sending and receiving Bibles to and from the remote repository.

A backup assistant provides various sorts of backups.

A restore assistant helps with restoring backups.

Simplified selection of Bibles for inclusion in the Parallel Bible.

A bunch of bugs was fixed.

One central Export Assistant handles all exports.

An OSIS file can be created in the variant that is fit for the Go Bible Creator.

It can create a Go Bible for use on a cell phone.

A Resource can have a variable home page that depends on the book now opened in the Editor.

Can backup and restore Resources.

Minor improvements here and there.

The scrolling position of the merge window sticks even during edits.

Communication with BibleTime over dcop was disabled. New system not yet implemented.

The temporal directory was moved back to where it belongs, in /tmp/bibledit.

When exporting books in USFM format, the user can select the desired books. And the files are now numbered.

Bulk spelling check.

Export of keyterms and their renderings.

An improved system for checking keyterms.
