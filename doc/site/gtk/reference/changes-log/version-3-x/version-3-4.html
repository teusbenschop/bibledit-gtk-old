Version 3.4

29 September 2008

If the remote git repository is on a USB drive, and is not available, then it does not keep retrying.

Another stab was given to the problem that the window becomes wide, hopefully this time fatal.

Status information is properly shared within the collaborating team, project shared dictionaries too.

A couple of regressions were fixed.

The Ctrl-Z accelerator for Undo was fixed.

The verse tracker was improved.

Formatted project notes editor.

Spelling is checked sooner after any change in the editor.

A usfm-to-pdf formatter was added.

The non-free XEP formatter was removed.

A couple of improvements were made to the collaboration setup process.
