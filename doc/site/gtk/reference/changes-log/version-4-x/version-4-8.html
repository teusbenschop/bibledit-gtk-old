Version 4.8

20 June 2013.
GtkHtml has been phased out in favor of WebKit/GTK+.
Inserting standard texts into the project notes editor was removed.Leaner toolbar for the project notes editor.Support for accessing git repositories through the https protocol, as used on github.com and others.The Restore Assistant was fixed. It gives better feedback: The restore will take effect after restarting Bibledit-Gtk.When creating a new project while a non-editable Bible was focused, the new project is now editable.The progress dialog more realistically indicates the progress while adding book templates to the project.When adding a Bible book to the project, the first one added will be focused in the text editor.When deleting a book from the project, the first one of the remaining books will be focused in the text editor.The check for sentence structure was removed because it crashes without reason.Double Strong's references in the Related verses window have been split into two entries.
