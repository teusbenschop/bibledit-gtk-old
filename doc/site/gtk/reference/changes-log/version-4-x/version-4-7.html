Version 4.7

30 March 2013
Modifications to allow for a win32 build.Edited the keyterm check so verses without renderings are highlighted.Fixed: The floating windows cannot be re-sized anymore on the newest Gnome.When importing the Aleppo Codex .zip file it also looks for the zip file in ~/Downloads.
Fixed: Importing the Aleppo Codex from Mechon-Mamre left out the last verse of each chapter.Importing text from BibleWorks with a byte order mark has been fixed.
There's a progress bar when importing USFM files.Added progress bar during removal of books from project.Added the option to compare another project to the OT and NT parallel passage checks. The user can choose to have a second project displayed beneath the one they are checking in the Parallel Passage check dialog.
