Edit

Edit dictionary

This dialog is accessible through menu File / Project / Properties, button Dictionaries, button Edit.

In this dialog the user can edit the contents of an editable dictionary.

Text can be added to this dialog, or removed.

Button "Sort, remove duplicates" does what it says, it sorts all the words in the order that the dictionary engine would order them, and it removes any duplicates.

The user can import words from another dictionary, or export them to another dictionary.

The user also can import from a file, or export to a file.

Whenever an import is done, the result is sorted and duplicates are removed.

The user can count the words in the dictionary.
